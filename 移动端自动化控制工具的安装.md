# Android 开发工具使用
#### 什么是adb
* adb ( Android Debug Bridge)是一个通用命令行工具，其允许您与模拟器实例或连接的 Android 设备进行通信。它可为各种设备操作提供便利，如安装和调试应用。
可以在命令行cmd 直接输入adb，看见有输出，说明环境变量的配置是正常的
##### 启动adb的服务器：
```
adb start-server
需要手机开启开发者模式，并允许usb调试
需要连续点击版本号：进入开发者模式，打开usb调试
```
